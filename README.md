# EnergyManeuverability_ThrustDragID
System identification code to identifiy a parametric thrust and drag model of a fixed-pitch propeller driven aircraft.
Time histories of body fixed acceleration, airspeed, angle of attack, sideslip, control surface deflections, and propeller angular velocity are needed in addition to vehicle properties. 
Least-squares, stepwise regression, and convex optimization solutions are provided in addition to estimated identified model uncertanities. See "Parametric Thrust and Drag Identification From Flight Test via Energy Maneuverability," AIAA 2026-4164. AIAA AVIATION 2026 Forum. by Sam Jaeger, et. al. for more information on the methods. To get started, refer to the GitHub wiki page.
