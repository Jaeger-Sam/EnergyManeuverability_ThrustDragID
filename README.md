# EnergyManeuverability_ThrustDragID
System identification code to identifiy a parametric thrust and drag model of a fixed-pitch propeller driven aircraft.
Time histories of body fixed acceleration, airspeed, angle of attack, sideslip, control surface deflections, and propeller angular velocity are needed in addition to vehicle properties. 
Least-squares, stepwise regression, and convex optimization solutions are provided in addition to estimated identified model uncertanities.
